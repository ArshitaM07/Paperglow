# 📖 PaperGlow

PaperGlow is a modern note-taking web application built with React and Vite. It provides a clean writing experience with note organization features and persistent local storage.

## ✨ Features

* Create new notes
* Edit note titles and content
* Delete notes
* Search notes instantly
* Pin important notes
* Theme switching (Dark, Light, Vintage)
* Automatic local storage persistence
* Responsive and clean UI

## 🛠️ Built With

* React
* Vite
* JavaScript
* Tailwind CSS

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/PaperGlow.git
```

### Navigate to the project

```bash
cd PaperGlow
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
